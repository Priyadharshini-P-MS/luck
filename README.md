# Witness Archive: Public Testimonies of ICE Enforcement in Chicago

This project curates a dataset of 500+ real narratives documenting the emotional and thematic impact of ICE operations in Chicago.

## Features
- Streamlit dashboard for exploring testimonies
- Filters by emotion, theme, source, and date
- Plotly charts and quote viewer
- Ethically sourced with public links

## Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
